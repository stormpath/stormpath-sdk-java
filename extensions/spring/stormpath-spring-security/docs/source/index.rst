|project|
=========

|project| allows you to use Stormpath almost instantly in any Spring Security application. It makes it *incredibly* simple to add user management, authentication and authorization to your application.

|project| helps automate all user registration, login, authentication and authorization workloads.  It is very flexible - use only what you need or leverage the entire feature set.

And the best part? **You don't even need a database!**
